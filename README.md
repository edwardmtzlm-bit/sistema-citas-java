# Sistema de Citas Java

Sistema de administración de citas desarrollado en Java utilizando Programación Orientada a Objetos.

# Instalación y configuración

1. Clonar el repositorio.
2. Abrir el proyecto en IntelliJ IDEA.
3. Ejecutar el archivo Main.java.
4. Contar con JDK 11 o superior.

# Uso del programa

El sistema permite:

* Registrar doctores.
* Registrar pacientes.
* Crear citas médicas.
* Validar acceso mediante administrador.
* Mostrar información de citas en consola.
