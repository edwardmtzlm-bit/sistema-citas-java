# Sistema de Citas Java

## Descripción

Sistema de administración de citas médicas desarrollado en Java como proyecto académico. El sistema permite registrar doctores, pacientes y citas médicas mediante una interfaz de consola, almacenando la información en archivos CSV para su posterior consulta.

## Funcionalidades

* Inicio de sesión mediante usuario y contraseña.
* Registro de doctores.
* Registro de pacientes.
* Creación de citas médicas.
* Almacenamiento de información en archivos CSV.
* Lectura de archivos CSV desde el sistema.
* Menú interactivo en consola.

## Estructura del Proyecto

### Modelos

* Administrador.java
* Doctor.java
* Paciente.java
* Cita.java

### Utilidades

* ArchivoCSV.java

### Base de datos

Carpeta `db`:

* doctores.csv
* pacientes.csv
* citas.csv
